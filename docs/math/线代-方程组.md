# 线性方程组


## 1.基础知识

0.克拉默法则 P243

1.系数矩阵只能做行变换，不能做列变换

2.齐次线性方程组具有非零解，充要条件是系数行列式|A|=0或者r(A)<n

3.齐次方程基础解系的个数=n-r

2.非齐次方程有唯一解的条件是|A|!=0 P245

3.基础解系线性无关。s(基础解系个数)=n(未知数个数)-r(系数矩阵的秩)

4.齐次线性方程Ax=0一定有解。至少有0解

5.求向量组的极大线性无关组时，只能都做行变化（后者列 变换）P242 例13
