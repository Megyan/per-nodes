## 3.比较数字的两种方法

* [] `内部编写比较逻辑.[]中括号两边一定要有空格`
* &&  || 连接符号 ` [ 2 -ne 3 ] && echo 0 || echo 1 表示2是不是不等于3，不等于输出0，等于输出1`

* 比较符号

```
-eq：等于  -ne：不等于 -lt：小于 -gt：大于 

-ge：大于等于 -le：小于等于 
```
